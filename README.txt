YESAYA ARYA - PORTOFOLIO STATIS (HTML)
---------------------------------------

FILE:
- index.html  -> halaman utama
- profile.jpg -> ganti dengan foto Anda (nama file harus tetap 'profile.jpg')
- cv.pdf      -> CV Anda (sudah disertakan)
- README.txt  -> file instruksi cepat ini

CARA PREVIEW DI KOMPUTER:
1) Klik dua kali file index.html untuk melihat tampilan di browser.
2) Untuk ganti link GitHub, buka index.html dengan Notepad, cari 'your-github', lalu ubah.

UPLOAD KE GITHUB PAGES (tanpa coding):
1) Buat akun github.com (jika belum punya).
2) Buat repository baru bernama: USERNAME.github.io (ganti USERNAME dengan username GitHub Anda).
3) Klik 'Add file' -> 'Upload files', drag & drop index.html, profile.jpg, cv.pdf, README.txt.
4) Klik 'Commit changes'.
5) Tunggu 1-2 menit, buka https://USERNAME.github.io/ (ganti USERNAME).

CATATAN:
- Ganti profile.jpg dengan foto Anda (ukuran persegi terlihat paling rapi).
- Jika situs belum tampil, buka 'Settings' -> 'Pages' -> pilih 'Deploy from a branch', Branch: main, Folder: /(root).
